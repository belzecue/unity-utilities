Localization Package
--------------------

Make sure that the README file has been updated to contain changes that may have been made.

To update the unity package for localization, select all files in the LocalizationPackage Folder and export using Assets > Export Package...
Add new localization package to shared location on the S Drive. 

iOS Requirements
----------------

Make sure that the README file has been updated to contain changes that may have been made

To update the iOSRequirements package, select the Editor Folder and the contained iOSRequirments folder and export using Assets > Export Package...
And the updated package to shared location on the S Drive.
